Breast Cancer Detection (Big Data Class Project)

This project uses the Breast Cancer Wisconsin (Diagnostic) Dataset to determine if a mass is benign or malignant using the scikit-learn library to train and test using multiple methods. The first method was the decision tree classifier, which is just training the model using the testing data and presenting it in a tree format. In the next part, we used a random forest map-reduce ensemble, which similarly to part one, trained 3 different trees of depths 3, 5, and 7 at once using multicore processing, then combined them using a majority vote and presented that tree as well as time, accuracy, sensitivity, and specificity.

In the final 2 parts, we trained the dataset on a support vector machine using the RBF, linear, and polynomial classifiers at once using multicore processing, then like the past parts combined them and presented the data as well as scatter plots of all 3 classifiers and the combined one. Finally, all of this data was used to predict if a mass is benign or malignant using any 2 features.

SKAM (WIP)

With just a tap of your phone with a friend (or search), you can share all your personal, work, or student information at once. Instantly get your new friend's name, phone number, emails, socials, and more on a virtual business card. No more asking them for each handle. Start up a conversation with your new acquaintance, get to know them more. You can choose how much you want to share with someone by simply choosing from the options student, work, and personal. Each of them contains specific socials and contact information depending on the environment that you met your new friend at.

This project was made with 3 other people, we used Firebase, React Native, and Expo.
